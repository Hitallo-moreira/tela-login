<h2 id="instalacao">Instalação</h2>

```
git clone git@github.com:Hitallo-moreira/tela-login.git
```
```
cd .\tela-login\
```

<b>2 - Instale as dependências:</b>

```
npm install
```

```
npm run dev
```

<b>Após isso basta pressionar o CTRL e clicar no endereço retornado no terminal:</b>
